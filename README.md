# COVID-19 Global Data Analysis

## Overview

This project focuses on analyzing the global impact of COVID-19 using real-world COVID-19 datasets. The objective of this project is to perform Exploratory Data Analysis (EDA), identify meaningful patterns and trends, and create an interactive dashboard for better visualization of insights.
The project was completed using Python for data analysis and Power BI for dashboard creation.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook
* Power BI

## Analysis Performed

* Global COVID Cases Trend Analysis
* Global Death Trend Analysis
* Top 10 Countries with Highest Cases
* Country-wise COVID Trend Analysis
* Vaccination vs Death Analysis
* Correlation Heatmap Analysis
* Death Rate Analysis

## Dashboard Includes

* Total Cases KPI
* Total Deaths KPI
* Countries Covered KPI
* Global Cases Trend
* Global Death Trend
* Top 10 Countries Analysis
* Vaccination vs Death Analysis

## Key Insights

* Global cases peaked in 2022
* Death rates were highest in 2021
* USA recorded the highest number of cases
* Increased vaccination showed a decline in death trends

## Conclusion

This project demonstrates how data analysis and visualization can be used to understand large-scale real-world health data and extract meaningful insights.
